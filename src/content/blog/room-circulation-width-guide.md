# 房間走道要留多寬？用門片、抽屜與椅子實測動線

速答：不要只量「家具邊緣到家具邊緣」的空白。真正需要保留的是人在使用家具時的**動態範圍**：房門要旋轉、衣櫃門要打開、抽屜要拉出、椅子要後退，人還要能上下床、彎腰整理與拿東西。先在平面圖畫出這些範圍，再用紙膠帶在現場走一次，會比套用一個固定走道數字可靠。

本文提供的是一般住宅初步規劃方法，不是各地建築、消防或無障礙規範。若空間涉及逃生、輪椅、助行器、固定裝修、兒童或長輩安全，應核對所在地規定並請合格專業人員確認。

## 先分清楚「靜態外框」與「使用範圍」

床、桌、衣櫃在商品頁上通常只有長、寬、高，但人使用時需要更多空間。衣櫃深 60 公分不代表牆前只要留 60 公分；若是外開門，還要加門片旋轉區。書桌深 60 公分也不代表占地只有 60 公分；椅子坐下、後退與人從背後通過都需要空間。

畫圖時可把每件家具分成兩層：

1. **家具外框**：真正落地、不能被其他家具占用的範圍。
2. **使用範圍**：門片、抽屜、椅子、人的腳與身體移動會經過的範圍。

兩個使用範圍可以偶爾重疊，例如沒有人坐書桌時可以打開旁邊櫃門；但每天同時發生的動作不能互相卡住。

## 四條最需要實測的路線

### 1. 房門到床邊

從門外開始，完整開門、進房、轉身、關門，再走到平常上下床的位置。檢查門片是否撞床角、門後是否堆物、夜間能否不繞過尖角。這條路也是緊急離開房間時最重要的路線，不要用收納籃、延長線或臨時紙箱占用。

### 2. 床邊到衣櫃

站在床邊走到衣櫃，打開最靠牆與最靠床的門或抽屜，模擬拿一件衣服、蹲下拿底層物品，再把門關上。如果必須先把床上物品移走才能開櫃，代表配置雖然「放得下」，但不適合每天使用。

### 3. 書桌椅的後退範圍

把椅子拉到實際坐姿，再往後退到能站起的位置。接著請另一個人從椅背後走過，或模擬自己拿書包、開門與移動垃圾桶。椅子後方若正好是主要走道，讀書與通行會互相干擾，應調整桌向或縮小桌深。

### 4. 窗戶、窗簾與清潔路線

窗邊不是只要能看到就算可用。要能拉窗簾、開窗、擦窗台、處理冷氣排水或除濕。高櫃若緊貼窗戶，可能讓採光、通風與清潔都變差。床靠窗時也要保留檢查潮濕與清潔的方式。

## 不用把單一尺寸當成通用答案

網路常出現「走道至少幾公分」的簡化答案，但房間用途與使用者差異很大。以下只能當作紙上配置的**測試起點**，不能取代現場、法規或專業判斷：

| 情境 | 可先在圖上測試 | 現場一定要做的動作 |
|---|---:|---|
| 偶爾通過的床側 | 約 55–60 公分起 | 側身、彎腰整理床單、夜間通行 |
| 每天反覆通過的主走道 | 約 70–80 公分起 | 雙手拿物、轉身、兩人錯身需求 |
| 衣櫃前 | 櫃門或抽屜深度＋站立區 | 打開最深抽屜、蹲下、拿衣服 |
| 書桌椅後 | 椅子坐姿＋站起範圍 | 後退、站起、背後通行 |
| 房門附近 | 完整門片旋轉區 | 從門外開門、進入、關門 |

如果使用者身形、行動能力、助行器或輪椅需求不同，起始數字就不應直接套用。最終判斷是「能否安全、穩定、重複完成日常動作」。

## 一個 280 × 340 公分房間的取捨案例

假設短牆有 90 公分房門，長牆有窗；需要單人床、120 公分書桌與 120 公分衣櫃。第一版把床與衣櫃排在同一長牆，紙上看起來還有空白，但衣櫃外開門會碰床角，書桌椅後退又占用入口。

第二版先把床沿無窗長牆放置，衣櫃移到門後不被門片掃到的位置，書桌靠近窗但讓螢幕與窗戶成側向。這樣做不一定最漂亮，卻把三種高頻動作分開：入口不碰椅子、衣櫃能完整打開、床邊有固定上下床位置。

可在 [房間家具配置工具](/zh/room-layout-planner/) 建立兩個版本，每次只改一件家具。完成後再用紙膠帶貼出床角、桌邊、衣櫃門片與椅子後退線，實際走五分鐘。

## 紙膠帶動線測試：15 分鐘完成

1. 在地面貼出床、桌、衣櫃外框，不必貼整面，只標四角與門片端點。
2. 用另一種顏色貼門片旋轉、抽屜拉出與椅子後退範圍。
3. 從門外開始，依序完成進門、放包包、開衣櫃、坐書桌、上下床、開窗。
4. 手上拿一個洗衣籃或書包再走一次，確認不是空手才勉強通過。
5. 關燈後用夜燈走一次主要路線，找出床角、線材與地面雜物。
6. 把每次停下、側身或移動物品的位置記在平面圖，再調整一件家具。

測試時不要因為「已經買了」就忽略不順。若大型家具尺寸不合，可比較窄桌、拉門衣櫃、床下收納或減少一件家具，而不是把所有走道壓到最低。

## 常見的五個錯誤

- 只量房間長寬，沒有量踢腳板、柱子、窗台與門框突出。
- 只畫櫃體，不畫門片和最深抽屜。
- 椅子全部推進桌下，沒有模擬有人坐著。
- 把臨時空白當成永久走道，之後又放垃圾桶、曬衣架或紙箱。
- 用民俗風水規則取代安全、通風、採光與逃生判斷。

## 完成前檢查表

- [ ] 房門、衣櫃門、抽屜與窗戶可以完整操作。
- [ ] 椅子有人坐時，主要入口仍能使用。
- [ ] 夜間從床到門不需要跨線材或繞過尖角。
- [ ] 常用物品可以站穩後拿取，不必踩椅子或長時間蹲在走道。
- [ ] 窗戶、窗簾、冷氣與插座仍可清潔和維修。
- [ ] 已用實際家具尺寸或包裝尺寸核對，不只看示意圖比例。

## 常見問題

### 小房間床邊只能留很窄，還能使用嗎？

要看那一側是否每天上下床、整理床單或通往窗戶。若只是低頻清潔側，可以把主要上下床位置留在另一側；若是唯一通道，應減少家具或改尺寸，不能只靠側身勉強通過。

### 衣櫃使用拉門就不需要前方空間嗎？

拉門不需要門片旋轉，但人仍要站立、蹲下、拿衣服，內部抽屜也可能向外拉。仍要用最深的使用狀態測試。

### 風水說走道要順，應該用什麼標準？

本站只把風水當民俗文化與舒適度參考。可觀察的標準是：門能開、人能走、家具能用、夜間不容易碰撞、通風與採光不被堵住；不保證任何運勢結果。

## 小結

房間走道不是一條固定寬度，而是一組每天重複的動作。先畫家具外框，再畫門片、抽屜、椅子與人的使用範圍；最後用紙膠帶完成進門、收納、讀書、上下床與清潔測試。能穩定完成這些動作的配置，才是真正「放得下也住得順」。
